==Description==

Panafacturas Factura Electrónica es un sistema que permite enviar factura electrónica al sistema tributario de la Dirección General de Ingresos (DGI) para ser registrados como lo estipulan las regulaciones presentes en la actual resolución. El servicio está diseñado para que los comercios que utilizan el plugin de Woocommerce puedan interconectarse con GTI y emitir factura electrónica.

Caracteristicas:

* Admite diferentes métodos de pago.
* Se adapta a cualquier tema de Wordpress.
* Imperceptible para el usuario.
* Multiusuario.

== Frequently Asked Questions ==

= ¿Hay algún requisito para poder utilizar el plugin? =

Sí, debe ser cliente de Panafacturas electrónica y configurar el sitio de manera correcta para su funcionamiento.

= ¿Existe alguna documentación de uso? =

Sí, puede acceder a la documentación por este [link](https://cdn.gticr.com/gticr/Documentos/Documentacion_Plugin_PanaFacturas.pdf) .

= ¿A que se debe el error? Ha ocurrido un error a la hora de generar su factura electrónica, por favor comuníquese con el administrador del sitio. =

Generalmente, esto se debe a una mala configuración en las credenciales del plugin, incorrecta configuración de atributos o impuestos de los productos.

= ¿Qué hacer si la factura electrónica no fue enviada al receptor.? =

* 1- Verificar que el correo ingresado sea correcto.
* 2- Verificar que todas las configuraciones sigan el estándar.
* 3- Comuníquese al correo integraciones@panafacturas.com.

= ¿Existe algún ambiente de pruebas.? =

Si, para solicitarlo escriba un correo a integraciones@panafacturas.com.

== Installation ==

Se puede instalar automáticamente desde la página de plugins de Wordpress.

Se puede instalar manualmente solicitando los archivos del plugin directo a GTI .
